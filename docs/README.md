### 1. 게임 시작
#### - 게임 시작 출력문
"경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)"

### 2. 각 자동차에 이름 부여하기
#### - 이름이 5자 이하인지 
#### - 알파벳이나 ',' 가 아닌 문자가 올경우 IllegalArgumentException 발생

### 3. 시도 할 횟수 입력
#### - 0~9 사이의 랜덤 값
#### - 0~9 사이 값이 아닐 경우 IllegalArgumentException 발생
#### - 랜덤 값이 4 이상일 경우에만 전진
#### - 시도 할 횟수 출력
"시도할 회수는 몇회인가요?"
#### - 매 횟수 마다 실행 결과 출력
실행 결과

woni :  <br/>

### 4. 게임 종료
#### - 우승자 출력
#### - 단독 우승자일 경우 
"최종 우승자 : pobi"
#### - 우승자가 여러명일 경우 ',' 를 이용하여 구분
"최종 우승자 : pobi, jun"
